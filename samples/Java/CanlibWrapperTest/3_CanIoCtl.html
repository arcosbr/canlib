CanIoCtl

This program shows how to use the canIoCtl method, as well as some of the other Canlib methods.

Using the wrapper

Source code

The various canIoCtl functions have been wrapped in various methods in the Handle class. The sample program demonstrates how these can be used.

Not using the wrapper

Please see the C# equivalent of this example. Aside from the fact that it uses out parameters instead of ByReference objects and some other language-specific details, the program is virtually identical to how this could have been done in Java.

The source code of the Handle class also shows how canIoCtl can be used.
